AI Engine — 6 Components
⚡ ImpactFlash AI Stack
Priority Scorer
XGBoost Regressor · 400 trees · 22 features
Scores every need 0–100 based on urgency, funding gap, beneficiaries, deadline, NGO track record, category weight and cross-features.
Fraud Detector
Isolation Forest · 200 trees · Unsupervised
Learns what normal submissions look like. Flags outliers without being told what fraud looks like. Combined with 6 auditable rule checks.
Score Explainer
SHAP TreeExplainer · Exact Shapley values
Tells admin exactly why each feature increased or decreased the score. Not approximation — mathematically exact contributions. Shown to admin only.
Allocation Optimizer
scipy.linprog · HiGHS solver · Linear Programming
Maximises total social impact for a given donation amount. Mathematically optimal — not proportional guesswork. Cause preferences as soft constraints.
Document Verifier
Claude Vision + PDF text extraction
Reads certificates, hospital bills, supporting docs. Checks: name match, number match, issuing authority, tampering signals, date consistency.
Feedback Loop
Outcome data → Model retraining
Every NGO outcome report updates trust scores and queues retraining data. Model gets smarter with every real delivery. Static rules never do this.

5-Layer Fraud Prevention
1
Identity Verification (Registration)
FCRA + PAN + Aadhaar + Bank penny drop. Government-verified identity. Can't fake a penny drop. Can't fake Aadhaar OTP.
2
AI Submission Fraud Detection
Isolation Forest detects abnormal patterns. 6 rule checks: high frequency, urgency without doc, amount spike, repeated escalations, cost per person anomaly, large request flag.
3
Self-Dealing Prevention (Donation)
Before every donation: donor PAN = NGO PAN? Same email? Donor is registered owner of NGO? → Blocked. 80G receipt also blocked. Network pattern analysis for circular rings.
4
Milestone Escrow (Post-Donation)
Money never released in full. 3 stages — each requires verified proof. If NGO disappears mid-campaign: funds frozen, donors notified, refund option given.
5
Admin Collusion Prevention
Two admins required above ₹1L. No single admin reviews same NGO repeatedly. All actions publicly logged. AI flags admin pattern anomalies. Rotating assignments.

What Happens If NGO Fails to Deliver
⏰ Missed Milestone
Admin notified. NGO given 7-day grace period. Trust score drops. If no response → funds frozen.
🚨 Fraud Detected
Funds frozen immediately. Admin investigates. NGO suspended. Donors notified. Refund option given.
❌ Proven Fraud
Full refund to donors. NGO permanently banned at individual level (Aadhaar blacklisted). FCRA violation reported.
